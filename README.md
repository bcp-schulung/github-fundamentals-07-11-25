# github-fundamentals-07-11-25

## Simple Python Web Application

This repository contains a simple Python web application that returns "Hello World" when accessed.

### Prerequisites

- Python 3.x
- pip

### Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

### Usage

1. Start the application:
```bash
python app.py
```

2. Access the application using curl:
```bash
curl http://localhost:5000/
```

The application will return:
```
Hello World
```

### Running the Application

The application runs on `http://0.0.0.0:5000` by default and can be accessed from:
- `http://localhost:5000/`
- `http://127.0.0.1:5000/`
